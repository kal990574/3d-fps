# 새 기능 구현 가이드

다음 기능을 구현하려고 합니다:
$ARGUMENTS

## 구현 가이드 요청사항

### 1. 설계 분석
- 기존 시스템과의 연관성 파악
- 필요한 클래스/인터페이스 설계
- SOLID 원칙을 준수한 구조 제안

### 2. 구현 단계
- 단계별 구현 순서 제시
- 각 단계별 필요한 코드 예시
- 의존성 및 참조 관계 설명

### 3. Unity 특화 고려사항
- 적절한 MonoBehaviour 생명주기 활용
- Inspector 노출 필드 설계
- 프리팹/씬 구성 제안

### 4. 테스트 방법
- 기능 테스트 방법 제안
- 엣지 케이스 고려

## 프로젝트 컨텍스트
- 현재 시스템: PlayerMove, PlayerStats, CameraRotate 등
- CharacterController 기반 이동
- 스태미나 시스템 존재
- DOTween 사용 가능

## 출력 형식
한국어로 상세한 구현 가이드를 작성해주세요.
