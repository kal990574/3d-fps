# 리팩토링 제안

다음 파일/코드를 리팩토링해주세요:
$ARGUMENTS

## 리팩토링 기준

### 코드 스멜 (Code Smell) 탐지
- 긴 메서드 (Long Method)
- 거대한 클래스 (Large Class)
- 중복 코드 (Duplicated Code)
- 매직 넘버/문자열
- 깊은 중첩 (Deep Nesting)
- 과도한 매개변수

### 리팩토링 기법 적용
- **Extract Method**: 긴 메서드 분리
- **Extract Class**: 책임 분리
- **Replace Magic Number**: 상수화
- **Introduce Parameter Object**: 매개변수 그룹화
- **Replace Conditional with Polymorphism**: 조건문 대체

### 보이스카우트 원칙
- 발견했을 때보다 더 깨끗하게
- 작은 개선도 적극적으로

## 출력 형식
1. **현재 문제점** 설명
2. **리팩토링 전** 코드
3. **리팩토링 후** 코드
4. **개선 효과** 설명

한국어로 작성해주세요.
