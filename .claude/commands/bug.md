# 버그 분석 및 수정

다음 버그를 분석하고 수정해주세요:
$ARGUMENTS

## 분석 프로세스

### 1. 버그 재현 조건 파악
- 어떤 상황에서 발생하는가?
- 재현 가능한 단계는?

### 2. 원인 분석
- 관련 코드 탐색
- 로직 흐름 추적
- 잠재적 원인 나열

### 3. 수정 방안
- 근본 원인 해결 방법
- 임시 해결책 (필요시)
- 사이드 이펙트 검토

### 4. 예방 조치
- 유사 버그 방지 방법
- 코드 개선 제안

## Unity 특화 버그 체크
- [ ] Null Reference Exception
- [ ] 컴포넌트 초기화 순서 문제
- [ ] 물리 연산 타이밍 (FixedUpdate vs Update)
- [ ] 코루틴 관련 이슈
- [ ] 이벤트 구독/해제 누락

## 출력 형식
1. **버그 원인** 요약
2. **수정 전** 코드
3. **수정 후** 코드
4. **수정 이유** 설명

한국어로 작성해주세요.
